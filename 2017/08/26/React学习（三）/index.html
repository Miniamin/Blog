二、webpack-dev-server

The webpack-dev-server is a little Node.js Express server。可以模拟一个服务器，底层是nodejs+express，让我们的前端代码可以不在阿帕奇中但是快速能够获得一个服务器环境。并且是自动检测代码改变的，可以替代之前的watch功能。
